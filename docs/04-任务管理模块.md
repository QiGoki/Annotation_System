# 任务管理模块

## 4.1 功能概述

任务管理模块负责标注项目的全生命周期管理。

### 核心功能

- 项目创建与配置
- 数据导入（批量导入待标注数据）
- 任务分发与分配
- 进度跟踪与统计
- 数据导出

### 用户角色与权限

| 功能 | 管理员 | 标注员 |
|------|--------|--------|
| 创建项目 | ✓ | ✗ |
| 编辑项目 | ✓ | ✗ |
| 删除项目 | ✓ | ✗ |
| 导入数据 | ✓ | ✗ |
| 分配任务 | ✓ | ✗ |
| 查看进度 | ✓ | ✗ |
| 导出数据 | ✓ | ✗ |
| 标注任务 | ✓ | ✓ |

**请求体**：

```json
        "label": "标注目标",
        "required": true

        "id": "classify_1",
        "type": "image_classify",
        "label": "目标类别",
        "required": true,
        "attributes": {
```


### 4.2.2 数据导入接口

#### POST /api/projects/{project_id}/import

批量导入数据

**请求体**（multipart/form-data）：
```
file: File (JSONL文件)
```

**JSONL文件格式**（每行一条数据）：
```jsonl
{"image_url": "/data/images/001.jpg", "width": 1920, "height": 1080}
{"image_url": "/data/images/002.jpg", "width": 1920, "height": 1080}
{"text": "今天天气很好。"}
```

**响应**：
```json
{
  "code": 200,
  "message": "成功导入100条数据，创建100个任务"
}
```

#### POST /api/projects/{project_id}/import-urls

通过URL列表导入

**请求体**：
```json
{
  "data_type": "image",
  "items": [
    {"image_url": "/data/images/001.jpg", "width": 1920, "height": 1080},
    {"image_url": "/data/images/002.jpg", "width": 1920, "height": 1080}
  ]
}
```

#### GET /api/projects/{project_id}/export

**查询参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|

### 4.3.2 数据导入示例

```python
# services/import_utils.py
import json
from typing import Iterator

def parse_jsonl(file_content: bytes) -> Iterator[dict]:
    """解析JSONL文件"""
    lines = file_content.decode('utf-8').strip().split('\n')
    for line in lines:
        if line.strip():
            yield json.loads(line)

def parse_json(file_content: bytes) -> Iterator[dict]:
    """解析JSON数组文件"""
    data = json.loads(file_content.decode('utf-8'))
    if isinstance(data, list):
        yield from data
    else:
        yield data
```

## 4.5 进度统计

### 4.5.1 统计指标

| 指标 | 说明 |
|------|------|
| 总任务数 | 项目下所有任务数量 |
| 待标注 | 状态为pending的任务数 |
| 标注中 | 状态为doing的任务数 |
| 已完成 | 状态为completed的任务数 |
| 完成率 | 已完成/总任务数 |

### 4.5.2 导出数据

- 支持导出已完成任务的标注结果
- 支持JSON、JSONL、CSV格式
- 导出内容包含：原始数据、标注结果、标注人、完成时间
